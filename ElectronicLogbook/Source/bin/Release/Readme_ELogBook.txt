This Readme file contains important information about the description of the
Electronic Logbook tool and its revisions.

-------------------------------------------------------------------------------


This file has been divided into the following sections for easy reference:

I.  Description of this tool Functionality
II. Revision History


I. Description
===============================================================================
===============================================================================

The Electronic Logbook Tool is designed to get Test Station and Test target 
configuration and record them every time. User can input and record other comments
also.
